# Workshop Booking Platform

## ✨ Design Principles
The original interface was minimal but lacked visual hierarchy and engagement. I redesigned the UI using a card-based layout, improved spacing, and clear call-to-actions to make the experience more intuitive for students.

## 📱 Responsiveness
I followed a mobile-first approach, ensuring layouts stack vertically, buttons are easily clickable, and typography remains readable across small screens.

## ⚖️ Trade-offs
I avoided heavy UI frameworks to maintain fast load times and better control over styling. Some advanced animations were intentionally limited to keep performance optimal.

## 🧩 Challenges
Understanding and restructuring the existing layout while preserving functionality was challenging. I approached this by modularizing components and progressively enhancing the UI.

## 🚀 Features
- **Hero Section**: Engaging headline, credibility text, and smooth scrolling CTAs.
- **Workshop Cards**: Interactive cards with hover animations and clear level badges.
- **Filter Bar**: Premium design with dynamic filtering.
- **Booking Form**: Clean, validated form with success feedback.
- **Dark Mode**: Toggle for light/dark themes.

## 🛠️ Tech Stack
- **React**: Functional components with hooks.
- **Tailwind CSS**: Utility-first styling.
- **JavaScript**: Dynamic interactivity.

## 💡 How to Run
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the development server: `npm start`
4. Open in browser: `http://localhost:3000`
